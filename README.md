> 版本: v0.2.1 | 日期: 2026-05-16

## 一、设计理念

### 核心理念

现在制作Galgame的东西（美术、音乐、配音）各家AI都做得很好了，但把这些拼装起来的过程太零碎。Galgame Studio 的定位是 **"傻瓜式打包机"** —— 你给我文本，我给你素材制作指南，你拿回来我帮你自动拼成一个能玩的游戏。

### 设计原则

| 原则           | 说明                                                         |
| -------------- | ------------------------------------------------------------ |
| **不造轮子**   | 不自己做AI绘图/AI音乐，只做提示词和流程引导                  |
| **本地优先**   | 剧本分析、提示词生成、项目打包全部本地完成                   |
| **API 不绑架** | 仅场景分析步骤需要用户自己配置的LLM API Key，支持所有OpenAI兼容接口 |
| **零命令行**   | Web可视化界面，4步向导，拖拽上传，点击生成                   |
| **轻量原则**   | 工具不生成任何内容，只输出"怎么生成"的指导和提示词           |
| **先导后包**   | 先引导用户去各AI工具制作素材，再本地打包成游戏               |
| **逐个导入**   | 每个提示词卡片自带素材导入按钮 → AI出图后直接拖进来自动命名分类 |

### 目标用户

- 有小说/故事，想做视觉游戏但没有编程能力的创作者
- 想做同人Galgame但被技术门槛卡住的人
- 想快速出原型的视觉小说团队
- 想做成人向/好感度向视觉小说的开发者

## 二、工作流程

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  ① 上传小说   │ ──▶ │ ② AI场景分析  │ ──▶ │ ③ 生成提示词  │ ──▶ │ ④ 打包导出   │
│  .txt / .md   │     │  小说→分镜     │     │  6类提示词    │     │  Ren'Py工程  │
│  选择模板      │     │  自动检测分支   │     │  +逐个导入素材 │     │  预览+素材检测 │
└──────────────┘     └──────────────┘     └──────────────┘     └──────────────┘
         │                    │                    │                    │
         ▼                    ▼                    ▼                    ▼
   自动识别章节          自动匹配对话归属        每张卡片自带导入按钮     导入的素材自动复制
   6套项目模板           角色+情绪+台词类型      切换模型自动填API地址    一键出完整项目
```

### 步骤详解

#### ① 上传小说

- 支持 `.txt` / `.md` 格式
- 自动检测章节分割（第X章 / Chapter X）
- 自动统计字数、段落数
- 拖拽或点击上传均可
- **6套项目模板可选**（校园/古风/科幻/悬疑/奇幻/成人恋爱）

#### ② AI 场景分析

- 将小说文本送入LLM（用户自备API Key）
- **切换模型自动填API地址**：选DeepSeek→自动填`api.deepseek.com`，选通义千问→自动填阿里dashscope，无需手动改
- 支持 GPT-4o / DeepSeek / Claude / 通义千问 / Groq
- LLM **自动匹配对话归属**，用户不需要手动指定谁说了什么
- 输出结构化的分镜脚本：
  - 场景划分、角色识别、台词归类（对话/旁白/内心独白）
  - 视觉指引（背景描述含构图光线）、CG建议
  - 氛围标签（BGM情绪、音效、转场）
  - 自动检测分支选择点

#### ③ 生成 AI 提示词 + 逐个导入素材 ★核心步骤

根据场景分析结果，生成7个标签页的完整制作指南，**每个提示词卡片自带素材导入按钮**：

| 标签页         | 内容                             | 导入素材                                         |
| -------------- | -------------------------------- | ------------------------------------------------ |
| 👤 **角色立绘** | 13种表情差分提示词               | 每个表情一个导入按钮 → 自动命名为`角色_表情.png` |
| 🖼️ **背景图**   | 场景背景提示词（自动去重）       | 每个地点一个导入按钮 → `bg_地点.png`             |
| 🎬 **事件CG**   | 关键场景插画提示词               | 每张CG一个导入按钮 → `cg_001.png`                |
| 🖥️ **标题/UI**  | 标题背景/Logo/对话框/名字框/按钮 | 每个UI元素一个导入按钮                           |
| 🎵 **BGM**      | Suno.ai提示词 + CC0资源链接      | —                                                |
| 🎙️ **配音**     | GPT-SoVITS / ElevenLabs两种方案  | —                                                |
| 🛠️ **工具推荐** | AI绘图/BGM/配音 付费+免费方案    | —                                                |

**素材导入工作流**：

```
AI出图 → 回到工具 → 点提示词旁边的"📤导入素材" → 选文件
→ 自动归类保存 → 按钮变"✅已导入" → 打包时自动复制到游戏目录
```

**操作区**：提示词底部可直接填项目名+素材目录，一键跳过素材直接打包。

#### ④ 打包导出

- 一键生成完整的 Ren'Py 项目（含分支选项的`menu:`块）
- **已导入的素材自动复制到项目目录**
- 自动生成5个文件
- 附带功能：**浏览器预览**（打字机/角色占位/选择支）、**素材检测**（百分比+缺失清单）

## 五、技术架构

### 技术栈

| 层           | 技术                              |
| ------------ | --------------------------------- |
| **前端**     | HTML + CSS + JavaScript (Vanilla) |


## 七、使用说明

### 快速开始

```bash
pip install -r requirements.txt
python main.py
# 浏览器访问 http://127.0.0.1:8888
```

### 完整使用教程

1. **上传小说** — 拖拽.txt文件，选择项目模板（可选）
2. **配置API** — 选择模型（自动填API地址），输入Key
3. **场景分析** — 点击开始，1-3分钟，自动匹配对话归属
4. **查看场景** — 检查分镜，注意🔀分支标记
5. **生成提示词** — 选画风，一键生成7个标签页
6. **复制提示词** — 去AI工具出图（推荐组合见下方）
7. **逐个导入素材** — 每个提示词卡片点📤，出图后直接导入，自动命名
8. **检查导入状态** — 所有卡片显示✅已导入即完成
9. **打包** — 第3步底部或第4步，输入项目名一键生成
10. **预览** — 点击浏览器预览看效果
11. **运行** — Ren'Py Launcher打开项目目录

## 八、FAQ

**Q: 对话归属要手动匹配吗？**
A: 不需要。LLM会自动从原文提取说话人和台词归属。只有原文写法极不清晰时才需要手动调整。

**Q: 怎么知道素材导入了没有？**
A: 每个提示词卡片旁边有按钮：📤=还没传 / ⏳=上传中 / ✅已导入=成功。打包时会自动复制。

**Q: 素材存哪了？**
A: `output/assets_staging/` 按类型分目录。打包时自动复制到Ren'Py项目的game目录。

**Q: 切换模型要不要手动改API地址？**
A: 不用。选DeepSeek自动填deepseek地址，选通义千问自动填阿里地址，选OpenAI自动填OpenAI地址。

**Q: 成人模板有什么特殊功能？**
A: 好感度系统（3女主追踪+阈值解锁）、画面震动（3种强度）、CG画廊（解锁式浏览）、好感度查看画面。选用此模板打包时代码自动注入。

**Q: 支持哪些LLM？**
A: 所有OpenAI兼容接口：GPT-4o、DeepSeek V3/R1、Claude 3.5 Sonnet、通义千问Plus、Groq Llama。

**Q: 没装Ren'Py怎么办？**
A: 从 https://renpy.org 下载。临时体验可用内置浏览器预览模式。
